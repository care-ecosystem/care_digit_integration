# Installation

## Stable release

To install Care DIGIT Integration, run this command in your terminal:

```sh
pip install Care DIGIT Integration
```

## From source

The source files for Care DIGIT Integration can be downloaded from the [Github repo](https://github.com/ohcnetwork/care_digit_integration).

You can either clone the public repository:

```sh
git clone git://github.com/ohcnetwork/care_digit_integration
```

Or download the [tarball](https://github.com/ohcnetwork/care_digit_integration/tarball/master):

```sh
curl -OJL https://github.com/ohcnetwork/care_digit_integration/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd care_digit_integration
pip install .
```
